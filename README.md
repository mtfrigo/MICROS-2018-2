# Setup

## Conectar na Galielo como root

`ssh root@galileoXX` 
`passwd`


## Adicionar novo usuario

`useradd -c <LOGIN> -s /bin/base -m <LOGIN>`
`passwd <LOGIN>` 

## Conectar com usuario

`ssh <LOGIN>@galileoXX`
`passwd <LOGIN>`

## Copiar FILE para a Galileo

`scp <FILE> <LOGIN>@galileoXX :`
